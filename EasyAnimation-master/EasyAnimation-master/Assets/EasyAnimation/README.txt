Version : 1.2

[增加] 新增“旋转效果”的基础控制，支持UI和GameObject
[修改] 修复“缩放效果”在多效果组合使用时可能会影响其他效果的控制
[增加] 为“缓动驱动器”增加直接对Vector3类型的直接控制，但性能消耗相对较float类型控制高

Version : 1.1

[移除] 移除GameObject控制类，现在不在支持控制GameObject对象
[修改] 修改动画控制组件的选项层级以及名称
[移除] 移除移动动画中的'反向'效果，现在统一使用新的'反转播放'效果
[移除] 移除多余的'缩放效果'类文件，现在放大缩小效果整合为'缩放效果'同时支持放大和缩小（这也是理所当然的正常逻辑）
[修改] 修改命名 ， 修复在重新激活播放时，位置错乱问题
[修改] 修改动画驱动计时逻辑，此前在不同设备上时间速率表现会受到帧率影响，新的逻辑不受帧率影响，但会出现低帧率的情况下动画不流畅.
